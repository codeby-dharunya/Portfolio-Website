# Personal Portfolio Website

This project is a simple personal portfolio website developed using HTML and CSS. It consists of two pages: Home and Contact. The Home page provides a brief introduction about me, while the Contact page contains the necessary details to connect with me.

The main objective of this project was to practice front-end web development concepts and enhance my skills in creating well-structured and visually appealing web pages.
This project represents one of my initial steps in my journey toward becoming a Full Stack Developer and reflects my enthusiasm for learning and building web applications.

## Features
• Home page with personal introduction  
• Contact page with contact information  
• Clean and user-friendly interface  
• Simple and organized project structure  
• Responsive design  

## Technologies Used
• HTML  
• CSS
